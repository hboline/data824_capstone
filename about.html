About the Data

The dataset used in this project is available from the UCI Machine Learning Repository. The abalone dataset contains physical attributes of abalone such as the length or weight of 4177 abalone. The purpose of this dataset is to make use of these attributes to predict the age of abalone. The age of an abalone can be found through a meticulous process involving cutting the shell through the cone, staining it, and counting the rings using a microscope. With the difficulty of this process, it is desirable to find a method to predict the age of the abalone accurately using other measurements that are more easily taken. The data comes from a 1994 paper The Population Biology of Abalone (Halitosis species) in Tasmania.

The abalone dataset has no missing entries and only required a minimal amount of manipulation to prepare. The measurements were multiplied by 200 (the original data had this data divided by 200), the age (num. rings + 1.5) was approximated as 1 + num. rings for nicer looking data, and filtered two entries with abnormally high height measurements leaving us with n = 4175 records in the data.

Variable Descriptions

  • Sex: Sex of abalone. Male, female, or "infant".
  • Height: Height of shell (mm).
  • Length: Longest shell measurement (mm).
  • Diameter: Perpendicular to length (mm).
  • Whole Weight: Weight of abalone whole (g).
  • Shucked Weight: Weight of abalone meat (g).
  • Viscera Weight: Weight of abalone guts after bleeding (g).
  • Shell Weight: Weight of abalone shell after drying (g).
  • Age: Age of abalone.

About the App

This application was built for the final project in the DATA 824: Data Visualization at the University of Kansas. The purpose of this app is to showcase various visualizations, interactions with data, and models on the data using R Shiny. The project repository is publically available on GitHub.

Tab Descriptions

  • Data Statistics: General statistics of abalone dataset. Statistical tests including 2-sample t-test between sex measurements & linear regression tests.
  • Data Exploration: Various visualizations. Box plots, frequency plots, violin plots, correlation plot.
  • Interactive Visualization: Interactive ggplot + plotly scatterplot with choice of variables & customization of visualization parameters.
  • Models: Linear regression for age prediction with interactive prediction & PCA on abalone dataset.
  • Data: Interactive data table.
